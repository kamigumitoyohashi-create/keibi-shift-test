データのつながり設計

1. 隊員スマホ隊員本人が暗証番号で入り、希望休または当日の残業時間を送信します。
2. クラウド保存本番では送信内容をクラウドDBに保存します。保存項目は隊員ID、現場ID、対象日、申請種別、値、メモ、送信日時、状態です。
3. 管理者タブレットこの画面で未確認・承認・差戻しを操作します。承認したものだけシフトや勤怠に反映します。
4. 反映先希望休は希望休データとシフト表へ、残業は勤怠カードの該当日の残業欄へ入ります。原本帳票の形は変えません。

CSV出力プレビュー

希望休入力

隊員本人がタブレットで自分の希望休を入力する画面です。休みたい日を押して「希望休」にしてください。
有給は管理者が「対象月・希望休」から入力してください。

現場設定

シフト表を切り替える現場名を設定します。ここで増やした現場は、上部の「現場」から選べます。

勤務記号設定

勤務記号・勤務時間・翌日休み・色・紐づく現場を変更できます。
自動シフト作成は現在の工場ルールに合わせているため、P/V/P5/V5/P17/V17/P15/V15などを削除すると自動配置が崩れる場合があります。まずは時間・色・表示名の変更から使うのが安全です。

有給・休暇管理

対象月を選ぶと、期間は自動で「前月21日〜当月20日」になります。 希望休・有給は、隊員ごとのカレンダーで日付をクリックして選択します。

月別保存・過去月読込

現在表示中の対象月を保存できます。保存しておくと、あとから過去月のシフトを読み込んで確認できます。
月を切り替える前に「この月を保存」を押しておくと安心です。

勤怠カード出力

個人別の勤怠・残業入力画面です。閲覧モードでは確認のみ、管理者編集モードでは残業や交通費を入力できます。
画面用は入力しやすく、印刷時は原本に近い勤怠カード形式で出力します。

印刷用シフト表

シフト表の一覧確認・印刷用画面です。閲覧モードでも確認できます。

隊員条件設定

ここで条件を変えられます。たとえば「大竹さんを230H→220H」「鈴木さんを120H→150H」「山本さんの後番を許可」などができます。
新規入社は「隊員を追加」、退社は各隊員カード右上の「退社削除」を使ってください。同期先や固定勤務の設定を変えた場合は、保存後に仮シフト作成で確認してください。
